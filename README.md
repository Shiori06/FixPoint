# FixPoint

設問1→problem1.py  
設問2→problem2.py  
設問3→problem3.py  
設問4→problem4.py  
テストデータ→monitoring.log  

テストデータとpythonファイルを同じフォルダに置き、コマンドライン上で 
`$python ファイル名.py`
とすることで実行できる。  
設問2以降は設問内容に応じてコマンドライン引数をパラメータとして受け取ることができる。  
指定するパラメータは設問中のN,m,tの順に対応している。  
例  
`$python ファイル名.py 3 2 30`
このときN=3, m=2, t=30となる。
